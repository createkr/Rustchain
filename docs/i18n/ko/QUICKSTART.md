# 🚀 빠른 시작

## 1. 설치

```bash
pip install clawrtc
```

## 2. 지갑 생성

```bash
# 새 RTC 지갑 생성
clawrtc wallet new

# 또는 기존 지갑 가져오기
clawrtc wallet import --private-key YOUR_KEY
```

## 3. 채굴 시작

```bash
# 단일 코어로 채굴 시작
clawrtc mine start

# 또는 모든 코어 사용
clawrtc mine start --threads auto
```

## 4. 잔액 확인

```bash
clawrtc balance
```

## 5. RTC 전송

```bash
clawrtc transfer --to RTC_DESTINATION_ADDRESS --amount 10.0
```

**지원하는 언어**: Python 3.8+
**지원하는 플랫폼**: Linux, macOS, Windows, PowerPC G3/G4/G5
